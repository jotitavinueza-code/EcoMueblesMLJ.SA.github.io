# 🌿 ECO MUEBLES MLJ S.A. — Sitio Web Estático ## Archivos del proyecto ``` ecomuebles/ ├── index.html ← Página principal (sitio web) ├── styles.css ← Estilos de la página principal ├── main.js ← Lógica del sitio (carga datos, animaciones) ├── admin.html ← Panel de administración ├── admin.css ← Estilos del panel admin ├── admin.js ← Lógica del panel admin └── README.md ← Este archivo ``` --- ## 🚀 Cómo usar el sitio ### 1. Abrir el sitio Haz doble clic en `index.html` para abrir el sitio en tu navegador. ### 2. Panel de Administración Haz clic en el botón **Admin** en el menú de navegación, o abre directamente `admin.html`. Desde el panel Admin puedes: - ✅ Cambiar teléfono, correo y ubicación - ✅ Agregar tus URLs de Instagram, Facebook y WhatsApp - ✅ Subir imágenes: hero, nosotros, logo, productos - ✅ Gestionar la galería de fotos (agregar/eliminar) > **Nota:** Los datos se guardan en el navegador (localStorage). Funcionan en el mismo computador donde los configuras. --- ## 📁 Agregar imágenes estáticas (opcional) Si quieres que las imágenes funcionen sin el panel admin, puedes colocar archivos en las siguientes rutas: | Imagen | Ruta | |--------|------| | Logo | `logo.png` | | Imagen hero | `hero-product.jpg` | | Sobre nosotros | `about-us.jpg` | | Silla | `productos/silla.jpg` | | Mesa | `productos/mesa.jpg` | | Puff | `productos/puff.jpg` | | Macetero | `productos/macetero.jpg` | | Personalizado | `productos/personalizado.jpg` | --- ## 🌐 Publicar en internet (gratis) ### Opción 1 — Netlify Drop (más fácil) 1. Entra a https://netlify.com/drop 2. Arrastra la carpeta `ecomuebles/` completa 3. ¡Tu sitio estará en línea en segundos! ### Opción 2 — GitHub Pages 1. Crea un repositorio en GitHub 2. Sube todos los archivos 3. Ve a Settings → Pages → Source: main branch 4. Tu sitio estará en `https://tuusuario.github.io/ecomuebles` --- ## 📞 Secciones del sitio 1. **Inicio (Hero)** — Frase impactante + imagen + botones CTA 2. **Sobre Nosotros** — Historia y valores 3. **Misión / Visión / Objetivo** 4. **Productos** — 5 tarjetas de productos 5. **Proceso** — 5 pasos del proceso productivo 6. **Impacto Ambiental** — 3 pilares + banner 7. **Galería** — Fotos de trabajos 8. **Contacto** — Datos + formulario + redes sociales 9. **Footer** — Links y redes --- ECO MUEBLES MLJ S.A. — Diseño que cuida el planeta 🌿
